## Formatting

To format, use VSCode Prettier extension, or run:
`npx prettier --write .`

To check it: `npx prettier --check .`

Formatting in checked in CI

## Start the Tailwind CLI build process

To build and watch the CSS, use the following command:
`npm run dev`
